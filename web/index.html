Politica de Privacidade

Quais dados coletamos

Nome completo, endereco de email e, opcionalmente, perfis de redes sociais (LinkedIn, Twitter, Instagram, GitHub) e links personalizados fornecidos voluntariamente pelo participante.

Para que usamos

Exclusivamente para geracao do cracha de identificacao em eventos. Os dados sao armazenados para permitir reimpressao e controle de participantes.

Base legal (LGPD art. 7, I)

O tratamento e realizado com base no consentimento expresso do titular, conforme a Lei Geral de Protecao de Dados (Lei 13.709/2018).

Seus direitos

Voce pode solicitar acesso, correcao ou exclusao dos seus dados a qualquer momento pelo email privacidade@gbuck.dev. Os dados serao removidos em ate 15 dias uteis (LGPD art. 18).

Retencao

Os dados sao mantidos pelo periodo do evento e ate 90 dias apos seu encerramento, sendo entao excluidos permanentemente.

Compartilhamento

Os dados nao sao compartilhados com terceiros, vendidos ou utilizados para fins de marketing.

Seguranca

Os dados sao armazenados em banco de dados com conexao criptografada (TLS). O acesso e restrito ao responsavel pelo sistema.
